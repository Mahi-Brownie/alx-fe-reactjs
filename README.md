# GitHub User Search Application

This project is a React application built using Vite.  
It allows users to search for GitHub profiles using the GitHub API.

## Features
- Search for any GitHub username
- Display user information including avatar, bio, and profile link
- Clean folder structure with reusable components
- API calls handled using Axios

## Technologies Used
- React (Vite)
- Axios
- JavaScript (ES6)
